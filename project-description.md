# MIDIU

## Editor de música asistido por IA que potencia la creación musical del artista.

La creatividad es rápida, pero los software de edición no, muchos clicks, menus y navegación: todo genera fricción, interrumpe el flujo, enfría la inspiración. Es por esto que les presentamos MIDIA, el nuevo Cursor para músicos, diseñado específicamente para expandir y potenciar las capacidades creativas.
